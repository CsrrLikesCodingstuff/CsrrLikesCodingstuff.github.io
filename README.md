# 🎯 Click Score Game

A simple and fun browser game built with HTML, CSS and JavaScript.

## 🎮 How to Play
- Click the button to increase your score
- Reach 10 points to win!

## 🛠️ Built With
- HTML
- CSS
- JavaScript

## 🚀 How to Run
1. Download the files
2. Open index.html in your browser
3. Start clicking and have fun!
